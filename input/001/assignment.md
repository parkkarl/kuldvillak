# JavaScripti kalkulaator

## Eesmärk

Luua brauseris töötav kalkulaator, mis suudab teha nelja põhitehet (liitmine, lahutamine, korrutamine, jagamine).

## Nõuded

- Lehel on kaks sisendvälja kahe arvu jaoks.
- Lehel on neli nuppu: `+`, `−`, `×`, `÷`.
- Pärast nupule vajutamist kuvatakse tulemus eraldi alal.
- Jagamine nulliga ei tohi rakendust katki teha — kuvada tuleb arusaadav veateade.
- Sisendi tühjuse korral peab kasutaja saama tagasiside, mitte `NaN`.
- Kasutaja sisend tuleb teisendada arvuks (`Number()` või `parseFloat`).
- Lahendus peab koosnema kolmest failist: `index.html`, `style.css`, `script.js`.

## Hindamiskriteeriumid

- Kõik neli tehet töötavad korrektselt.
- Vealoogika (jagamine nulliga, tühi sisend) on kaetud.
- HTML on semantiline, CSS on eraldi failis, JS on eraldi failis.
- Kood ei sisalda kõvasti kodeeritud testandmeid.
